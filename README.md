# ⚽ Football Manager - Училищен проект (11. клас)

[cite_start]Настолно Windows Forms приложение за цялостно администриране на футболни първенства, клубове, играчи и трансфери [cite: 5-8]. [cite_start]Проектът включва автоматично генериране на футболна програма, изчисляване на класиране и извеждане на статистики [cite: 8-11].

## 🛠 Използвани технологии
* [cite_start]**Език:** C# (WinForms) [cite: 2]
* [cite_start]**База данни:** Microsoft SQL Server (T-SQL) [cite: 77]
* **Връзка с БД:** ADO.NET (`System.Data.SqlClient`)
* **Дизайн:** Custom Dark Theme (програмно генериран UI)

## 🌟 Основни функционалности (Модули)
* [cite_start]**Вход и Роли:** Три нива на достъп (Администратор, Оператор/Съдия, Потребител/Фен) [cite: 12-16].
* [cite_start]**Клубове и Играчи (CRUD):** Картотекиране на футболисти и управление на отбори [cite: 19-31].
* [cite_start]**Трансферен център:** Осъществяване на трансфери между клубовете с пазене на пълна история [cite: 32-37].
* [cite_start]**Първенства и Програма:** Автоматично генериране на мачове на принципа "всеки срещу всеки" [cite: 41-46].
* [cite_start]**Резултати и Събития:** Въвеждане на голове и минути за конкретни играчи в даден мач [cite: 47-52].
* [cite_start]**Класиране:** Автоматично изчисляване на точки (3 за победа, 1 за равен) и голова разлика [cite: 59-63, 137-141].
* [cite_start]**Статистика:** Извеждане на Топ Голмайстори [cite: 70-72].

## 🚀 Ръководство за инсталация и стартиране

За да стартирате проекта на ваш компютър, следвайте тези стъпки:

### 1. Подготовка на Базата данни
Проектът изисква инсталиран **Microsoft SQL Server** (или LocalDB чрез Visual Studio).
1. Отворете SQL Server Management Studio (SSMS) или Server Explorer във Visual Studio.
2. Създайте нова база данни с име `FootballManagerDB`.
3. [cite_start]Отворете файла `schema.sql` (намира се в главната директория) и го изпълнете, за да създадете таблиците[cite: 108].
4. [cite_start]Отворете файла `seed.sql` и го изпълнете, за да напълните базата с първоначални тестови данни (клубове, играчи, лиги)[cite: 108].

### 2. Настройка на връзката (Connection String)
1. Отворете проекта във **Visual Studio**.
2. Намерете файла `Db.cs`.
3. Променете променливата `connectionString` така, че да сочи към вашия SQL сървър. 
   * *Пример за LocalDB:* `Server=(localdb)\MSSQLLocalDB;Database=FootballManagerDB;Integrated Security=True;`
   * *Пример за SSMS:* `Server=.;Database=FootballManagerDB;Integrated Security=True;TrustServerCertificate=True;`

### 3. Стартиране
Уверете се, че NuGet пакетът `System.Data.SqlClient` е инсталиран. Натиснете **Start (F5)** във Visual Studio.
